# Coloque aqui os ícones do aplicativo

## Arquivos necessários:

- `icon.ico` - Ícone principal do Windows (256x256 ou maior)
- `icon.png` - Ícone alternativo (opcional)

## Onde conseguir ícones:

1. **Criar online**: https://favicon.io/ ou https://www.icoconverter.com/
2. **Banco gratuito**: https://www.flaticon.com/ ou https://icons8.com/
3. **Criar com IA**: https://www.logoai.com/ ou https://www.canva.com/

## Dicas:

- Use resolução alta (pelo menos 256x256)
- Formato .ico suporta múltiplas resoluções
- Mantenha o design simples e reconhecível
- Use cores que representem o projeto

## Temporário:

Se não tiver um ícone ainda, o Electron usará o ícone padrão.
O build funcionará normalmente sem o ícone customizado.
